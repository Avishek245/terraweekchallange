# 🚀 TerraWeek Challenge 2026

Welcome to my **TerraWeek Challenge 2026** repository! 🎉

This repository documents my journey through the **7-Day TerraWeek Challenge** organized by **TrainWithShubham**, where I learned and implemented **Infrastructure as Code (IaC)** using **Terraform**.

Throughout this challenge, I explored Terraform from the basics to advanced concepts, provisioned AWS infrastructure, implemented reusable modules, managed remote state, wrote tests, performed security scanning, and built a complete Terraform capstone project.

---

## 📌 Challenge Overview

The TerraWeek Challenge is a hands-on learning program designed to help engineers build real-world Terraform skills through daily practical tasks.

During this challenge, I learned:

- ✅ Infrastructure as Code (IaC)
- ✅ Terraform Fundamentals
- ✅ HashiCorp Configuration Language (HCL)
- ✅ Providers & Resources
- ✅ Variables, Outputs & Locals
- ✅ AWS Infrastructure Provisioning
- ✅ Terraform State Management
- ✅ Remote Backend using Amazon S3
- ✅ Terraform Modules
- ✅ Workspaces
- ✅ Terraform Testing
- ✅ Security Scanning with Trivy
- ✅ CI/CD Concepts
- ✅ Terraform Best Practices
